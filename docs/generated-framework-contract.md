# Generated Framework Contract

Every framework generated by `qa-framework-generator` must conform to this contract. The static validator (`validate_static`) and Maven validator (`validate_maven_compile`) enforce it automatically.

---

├── pom.xml                                        REQUIRED
├── README.md
├── .gitignore
├── .github/
│   └── workflows/ui-tests.yml
├── src/
│   ├── main/
│   │   └── java/<pkg_path>/
│   │       ├── base/
│   │       │   └── BasePage.java                  extends — generated page objects extend this
│   │       ├── config/

## Invariants

### Java

- All generated Java files start with `// Generated by qa-framework-generator` (added by file writer).
- All classes have explicit package declarations matching `<package_name>.<subpackage>`.
- No `Thread.sleep(...)` — static validator will flag it.
- No `TODO_REPLACE_LOCATOR` unless `--allow-placeholder-locators` is passed.
- Page objects extend `BasePage` and use `By` locators only (not PageFactory).
- Test classes extend `BaseTest`.
- Every `@Test` has `groups = {...}` — static validator will flag missing groups.
- Test classes live in `src/test/java/<pkg>/tests/`.
- Page objects live in `src/test/java/<pkg>/pages/` (Milestone 2).

### Maven

- `mvn -q -DskipTests compile` must exit 0 — validated by `validate_maven_compile()`.
- `mvn -q test -Dgroups=smoke -Dheadless=true` must exit 0 or fail only because the target site is unreachable (not a compilation or TestNG config issue).
- Environment is selected at runtime: `mvn test -Denv=qa`.
- Browser is selected at runtime: `mvn test -Dbrowser=firefox`.

### XML / YAML

- `pom.xml` and `testng.xml` must parse as well-formed XML — static validator strips `<!DOCTYPE>` before parsing.
- `.github/workflows/ui-tests.yml` must be valid YAML (Jinja-rendered, no markers remaining).

### Security

- No API keys, passwords, or secrets in any generated file.
- `OPENAI_API_KEY` is never written to generated Java code, CI YAML, README, or logs.

## Maven Command Reference

| Command | Purpose |
|---|---|
| `mvn -q -DskipTests compile` | Compile all sources — fastest check |
| `mvn test -Dgroups=smoke -Dheadless=true` | Run smoke tests headless |
| `mvn test -Denv=qa -Dgroups=regression` | Run regression against QA env |
| `mvn test -Dbrowser=firefox -Dheadless=true` | Run with Firefox |
| `mvn test -Dretry.count=2` | Retry flaky tests up to 2 times |

## Repair Hints

| Violation | Detected by | Fix |
|---|---|---|
| `pom.xml` missing | `check_pom_xml_exists` | Re-run renderer |
| Malformed XML | `check_xml_<file>` | Fix template or renderer context |
| `{{ var }}` in output | `check_jinja_markers_<file>` | Add missing variable to render context |
| `Thread.sleep` in Java | `check_no_sleep_<file>` | Repair node replaces with comment |
| `@Test` without `groups` | `check_testng_groups_<file>` | LLM repair or template fix |
| `TODO_REPLACE_LOCATOR` | `check_placeholder_locators_<file>` | LLM repair with real locator |
| `target/` in file list | `check_no_artifact_<file>` | Remove from generated file list |
